## Specs
<b>MCU</b> : STM32F446RCT6
<b>Number of Keys:</b> 64 Kailh choc hotswap sockets
<b>Number of LEDS:</b> 64 SK6812-mini e (1 per key) 
<b>Displays:</b> Adafruit 1.8" TFT Display
<b>Connection between halfs :</b> Serial over USB-C to USB-C cable
<b>Wire Debugger:</b> optional headers
